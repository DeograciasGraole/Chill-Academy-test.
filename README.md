  <h2>📋 Overview</h2>

  <h2>✨ Features</h2>
  <ul>
    <li>📱 Responsive layout with HTML and CSS</li>
    <li>🛍️ Product listing and detail views</li>
    <li>🛒 Shopping cart management (add, remove, update items)</li>
